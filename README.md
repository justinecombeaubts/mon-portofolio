# 🌟 Mon Portfolio - Justine Combeau

Bienvenue sur mon **portfolio personnel** ! Ce projet met en avant mes compétences en développement web, mes projets, mes loisirs et mon livre.

🔗 **[Voir le portfolio en ligne](https://justinecombeaubts.github.io/mon-portofolio/](https://justinecombeau-portofolio.netlify.app/))** *(à adapter si déployé ailleurs)*

---

## 📁 Structure du projet

```
mon-portofolio/
├── index.html          # Page principale du portfolio
├── assets/             # Ressources statiques (images, icônes, etc.)
│   └── images/
│       ├── favincon.ico
│       └── logo.png
├── styles/             # Feuille de style CSS
│   └── main.css
└── scripts/            # Scripts JavaScript
    └── main.js
```

---

## 🛠 Technologies utilisées

- **HTML5** : Structure de la page.
- **CSS3** : Styles et animations (avec [Font Awesome](https://fontawesome.com/) pour les icônes et [Google Fonts](https://fonts.google.com/) pour les polices).
- **JavaScript** : Interactivité (ex: basculer entre les thèmes clair/sombre, animations).
- **Google Analytics** : Suivi des visites (ID: `G-YM5GMY941R`).

## 🚀 Comment utiliser ce projet ?

### 1️⃣ **Ouvrir localement**
1. Cloner le dépôt :
   ```bash
   git clone https://github.com/justinecombeaubts/mon-portofolio.git
   ```
2. Ouvrir le fichier `mon-portofolio/index.html` dans un navigateur.

### 2️⃣ **Déployer sur GitHub Pages**
1. Pousser le code sur la branche `main`.
2. Activer **GitHub Pages** dans les paramètres du dépôt (Settings > Pages).
3. Sélectionner la branche `main` et le dossier `/mon-portofolio`.

---

## 🎨 Personnalisation

- **Modifier le contenu** : Éditer le fichier `mon-portofolio/index.html` pour mettre à jour les sections (À propos, Compétences, Projets, etc.).
- **Changer le style** : Modifier le fichier `mon-portofolio/styles/main.css`.
- **Ajouter des projets** : Ajouter des cartes dans la section `#projects` du fichier `index.html`.
